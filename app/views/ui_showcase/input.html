Input

Base input with labels, hints/errors, disabled state, and optional masks (number, currency, date).

Text

Default text field with a hint and tooltip.

{{ input( name='demo_text', label='Full name', placeholder='Jane Doe', hint='Use your legal name as it appears on documents.', tooltip_content='This is used for contracts and invoices.', type='text' ) }}

Textarea

Multiline input for longer descriptions.

{{ textarea( name='demo_textarea', label='Message', placeholder='Write your message...', hint='Tip: you can resize the textarea vertically.', tooltip_content='This is a multiline field using the shared input wrapper.' ) }}

Textarea (disabled)

Disabled styling with prefilled text.

{{ textarea( name='demo_textarea_disabled', label='Read-only message', placeholder='Write your message...', hint='This textarea is not editable in this state.', disabled=true, value='Locked: you can still read the message.' ) }}

Number (masked)

Allows only numeric input and formats with grouping spaces.

{{ input( name='demo_number', label='Units', placeholder='0', hint='Example: 12000', type='number', value=12000 ) }}

Currency (masked)

Useful when you want a visually grouped amount.

{{ input( name='demo_currency', label='Budget', placeholder='0', hint='Grouping is visual; submitted value matches what you typed.', type='currency', value='2500000' ) }}

Password

Includes a visibility toggle button.

{{ input( name='demo_password', label='Password', placeholder='••••••••', hint='At least 8 characters.', type='password' ) }}

Date (masked)

Simple dd/mm/yyyy mask (for display only).

{{ input( name='demo_date', label='Date', placeholder='dd/mm/yyyy', hint='Example: 18/03/2026', type='date', value='18/03/2026' ) }}

Disabled + error

Shows disabled styling and error state.

{{ input( name='demo_disabled', label='Disabled field', placeholder='Can’t edit', hint='This field is read-only in this state.', disabled=true, type='text', value='Locked' ) }} {{ input( name='demo_error', label='Email', placeholder='name@company.com', error_message='Please enter a valid email address.', type='text', value='not-an-email' ) }}

Date / Time Picker

Three variants returning ISO8601 values: date (`YYYY-MM-DD`), time (`HH:mm:00`), and datetime (`YYYY-MM-DDTHH:mm:00`).

Date picker

{{ datetimepicker( name='demo_date_picker', variant='date', label='Ngày', hint='ISO: YYYY-MM-DD', value='2026-03-23' ) }}

Time picker

{{ datetimepicker( name='demo_time_picker', variant='time', label='Giờ', hint='ISO: HH:mm:00', value='10:01:00' ) }}

Date time picker

{{ datetimepicker( name='demo_datetime_picker', variant='datetime', label='Ngày giờ', hint='ISO: YYYY-MM-DDTHH:mm:00', value='2026-03-23T10:01:00' ) }}

Date picker (month + year only)

Fix `day=1`, let users choose only month and year.

{{ datetimepicker( name='demo_date_picker_month_year_only', variant='date', label='Tháng áp dụng', hint='Selectable: year + month. ISO output is still full date.', value='2026-03-01', extra_attributes={ 'day': 1 } ) }}

Date picker (year only)

Fix `month=1` and `day=1`, let users choose only year.
